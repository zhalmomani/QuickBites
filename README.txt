QUICKBITES LANDING PAGE - CLIENT READY PACKAGE
==============================================

Thank you for choosing QuickBites for your food delivery startup website. This package includes all necessary files for a complete, modern, and responsive landing page.

FILE STRUCTURE:
---------------
/
├── index.html          (Main landing page)
├── about.html          (About Us page)
├── privacy.html        (Privacy Policy page)
├── terms.html          (Terms of Service page)
├── cookies.html        (Cookies Policy page)
├── css/
│   └── style.css       (Main stylesheet)
├── js/
│   └── script.js       (JavaScript functionality)
└── README.txt          (This file)

EDITING CONTENT:
----------------
1. TEXT CONTENT:
   - Open the HTML files in any text editor
   - Modify text within the HTML tags
   - For example, to change the headline, find <h1>Delicious Food Delivered Fast</h1> and replace with your desired text

2. IMAGES:
   - Replace image URLs in the src attribute of <img> tags
   - Current images are from Unsplash and are placeholders
   - Example: <img src="your-image-url.jpg" alt="Description">
   - Recommended image sizes:
     * Hero image: 1200x800px
     * Feature images: 600x400px
     * Team images: 300x300px

3. COLORS:
   - Primary color: #FF6B00 (Orange)
   - Secondary color: #FFC300 (Yellow)
   - Accent color: #2E2E2E (Dark Gray)
   - To change colors, edit the CSS file (css/style.css)
   - Search for color codes and replace with your preferred colors

4. LINKS:
   - Update navigation links in the <nav> section of each HTML file
   - Modify social media links in the footer section
   - Update contact information in the contact section

5. FORM ACTION:
   - The contact form currently shows an alert on submission
   - To integrate with a backend service:
     * Open js/script.js
     * Find the form submission handler
     * Replace the alert with your preferred form handling method

RESPONSIVENESS:
---------------
This website is fully responsive and will work on:
- Desktop computers
- Tablets
- Mobile devices

The design follows a mobile-first approach with breakpoints at:
- 992px (for tablets)
- 768px (for smaller tablets and large phones)
- 576px (for mobile phones)

ACCESSIBILITY:
--------------
- Proper contrast ratios for text
- Semantic HTML structure
- ARIA roles where appropriate
- Keyboard navigable

BROWSER SUPPORT:
----------------
This website works on all modern browsers:
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
